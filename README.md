# 真・三國無雙起源 — 夢幻四英傑全開存檔

Dynasty Warriors Origins (Steam) - Dream of the 4 Heroes DLC fully unlocked save file.

Steam 版本存檔，已解鎖「夢幻四英傑」DLC 全部內容。

## 安裝方式

將 `SLOT0008.dat` 複製到以下路徑：

```
%LOCALAPPDATA%\KoeiTecmo\Dynasty Warriors Origins\Savedata\<steam_user_id>\
```

`<steam_user_id>` 是你的 Steam 數字 ID，通常資料夾內只有一個子目錄。

> 請先備份你原有的存檔。使用本存檔造成的任何問題，風險自負。
